# History of Go

Go was created at Google in 2007 by Robert Grisemer, Ken Thompson, and Rob Pike to solve problems that they were facing 
at Google in their day-to-day work. 

Their goal was “to improve programming productivity in an era of multi-core networked machines and large codebases” 
(https://go.dev/solutions/google/), that is,  https://go.dev/talks/2012/splash.article


The uses cases listed on the Go website are 
cloud & network services
CLIs (Command-line Interfaces)
Web development
Development Operations & Site Reliability Engineering. 


The Go documentation has tutorials and resources listed to learn Go
https://go.dev/learn/
The Go website also lists some Go books as well, such as 
  The Go Programming Language 
  Get Programming with Go 
  Introducing Go
  Go Programming Blueprints
  Concurrency in Go
